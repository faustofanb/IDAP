## 📊 技术栈概览

### 前端 (Port 3000)

- Vue 3.4+ + Element Plus 2.5+
- Pinia + Vue Router
- Vite 5 + Bun
- ECharts 5

### Java Gateway (Port 8080)

- Spring Boot 3.2+ (JDK 21/25)
- Spring Security + JWT
- Spring WebSocket
- HikariCP + Resilience4j

### Python AI Engine (Port 8001)

- FastAPI 0.109+ + LangChain 0.1+
- LangGraph (Agent 编排)
- FAISS / Milvus (向量存储)

### 数据层

- PostgreSQL 16+ (主数据库)
- Redis 7+ (缓存)
- Milvus 2.3+ (向量数据库)

---

## 🔒 关键约束

1. **Python 不直接访问数据库** - 所有 SQL 执行由 Java Gateway 控制
2. **所有数据操作需经过白名单校验** - 防止 SQL 注入
3. **JWT Token 验证** - 所有 `/api/v1/*` 接口需认证
4. **审计日志** - 记录所有用户操作
5. **流式推送** - RAG 和 LLM 对话使用 SSE 流式返回
